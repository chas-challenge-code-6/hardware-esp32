Gas Task function. 

This function handles gas sensor operations in a Free\+RTOS task. It reads gas concentration data from the MQ2 sensor and sends it to a queue for processing. The task runs in an infinite loop, waiting for data to be available in the queue. When data is received, it is sent to the network for processing. The task uses the MQ2 sensor to measure gas concentration in parts per million (PPM) and sends the data to a queue.
