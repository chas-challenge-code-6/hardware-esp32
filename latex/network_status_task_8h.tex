\doxylink{class_network}{Network} Status Task Header File. 

This file contains the declaration of the network\+Status\+Task function, which is used to handle network status monitoring operations in a Free\+RTOS task. The task is responsible for checking the network connection status and updating the network event group accordingly. It also handles modem reconnection logic and manages the network event mutex to ensure thread safety.
