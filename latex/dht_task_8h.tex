\doxysubsection{Detailed Description}
DHT Task Header File. 

This file contains the declaration of the dht\+Task function, which is used to handle DHT sensor operations in a Free\+RTOS task. The task is responsible for reading temperature and humidity data from the DHT sensor and sending it to a queue for processing. The task runs in an infinite loop, waiting for data to be available in the queue.

DHT Task function. 

This function handles DHT sensor operations in a Free\+RTOS task. It reads temperature and humidity data from the DHT sensor and sends it to a queue for processing. The task runs in an infinite loop, waiting for data to be available in the queue. When data is received, it is sent to the network for processing.
