# **Hardware Design & Components**

## **Hardware Architecture**
Project Sentinel is built around a modular hardware architecture:

- **Sensor Subsystem:**
  - Gas Sensor (MQ-135) → analog input (ADC)
  - Temperature & Humidity Sensor
  - Heart Rate / SpO₂ Sensor


- **Processing & Control:**
